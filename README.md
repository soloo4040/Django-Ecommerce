# Django Ecommerce

A simple ecommerce web application built with HTML, CSS, BOOTSTRAP and DJANGO framework 

## Features
- Product listing
- Shopping cart
- User authentication
- Checkout system

## Technologies
- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap css

## Installation

git clone https://github.com/soloo4040/Django-Ecommerce-.git
cd Django-Ecommerce-
pip install -r requirements.txt
python manage.py runserver
